# AI MENTOR - Nauka AI od podstaw do PRO

Personalny tutor AI z **dwoma trybami pracy**: systematyczna nauka (Learner) + rapid building (Creator).

## 🎯 Co to jest?

System który uczy cię praktycznego wykorzystania AI przez:
- **Learner Mode** - systematyczny kurs od podstaw do mastery
- **Creator Mode** - szybkie testowanie narzędzi, budowanie portfolio, generowanie contentu
- **Personalizację** - wszystko dopasowane do Twojej branży i celów
- **Problem-first approach** - zawsze od problemu, nie od narzędzia

---

## 🚀 Jak zacząć? (3 kroki)

### 1️⃣ Stwórz swój profil (5 minut)

1. Wejdź do folderu **`01_utworz_profil`**
2. Otwórz plik **`prompt.txt`**
3. Zaznacz i skopiuj całą zawartość (Ctrl+A, Ctrl+C)
4. Wklej do ChatGPT/Claude/Gemini (nowy czat)
5. Odpowiedz na pytania AI
6. AI wygeneruje **`PROFIL.md`** - skopiuj i zapisz jako plik

**Co się dzieje:**
- AI zadaje pytania o Twoją pracę, cele, styl nauki
- Generuje spersonalizowany profil na podstawie odpowiedzi
- Ten profil będzie używany do dostosowania nauki

---

### 2️⃣ Stwórz projekt AI i dodaj pliki (3 minuty)

**ChatGPT Projects / Claude Projects / Gemini Gems:**

1. **Stwórz nowy projekt/Gem:**
   - ChatGPT: "Projects" → "Create project" → nazwa "AI Mentor"
   - Claude: "Create a project" → nazwa "AI Mentor"
   - Gemini: "Create a new Gem" → nazwa "AI Mentor"

2. **Dodaj pliki** z folderu **`02_pliki_projektu`**:
   - `MAIN_MENU.md` ⭐ (główny kontroler)
   - `mentor_learner.md` (tryb nauki)
   - `mentor_creator.md` (tryb tworzenia)
   - `PROFIL.md` (twój profil z kroku 1)

   **Jak dodać:**
   - ChatGPT: "Upload files" lub skopiuj treść
   - Claude: "Add content" → wklej treść każdego pliku
   - Gemini: Sekcja "Knowledge" → dodaj pliki

3. **Project Instructions:**
   - Otwórz plik `INSTRUKCJA_DO_PROJEKTU.md`
   - Skopiuj całą zawartość
   - Wklej do "Instructions" / "Custom instructions" projektu/Gema

---

### 3️⃣ Start!

Napisz w projekcie:

```
Cześć, zacznijmy naukę AI
```

AI:
- Wczyta Twój profil
- Pokaże główne menu
- Zaproponuje tryb (Learner/Creator)

**Wybierz:**
- `[L]` Learner - systematyczna nauka
- `[C]` Creator - budowanie, testowanie, content

Możesz przełączać tryby w dowolnym momencie!

## 🎯 Filozofia kursu

### Problem-First (NIE Tool-First!)

✅ **Nasz approach:**
1. Zdefiniuj problem biznesowy
2. Policz koszt problemu (ROI)
3. Czy AI pomoże? (często tak, czasem nie)
4. Wybierz narzędzie do problemu

### Integration > Tools

Narzędzie to środek, nie cel.

**Uczymy:**
- Jak AI wpasować w workflow
- Jak zmierzyć wartość
- Kiedy użyć, kiedy nie
- Human-in-the-loop

### Real ROI

Każde rozwiązanie = mierzalna wartość:
- "2h dziennie saved = 40h/miesiąc"
- "$5k saved (Lovable vs developer)"
- "10 klientów dzięki portfolio"
